ページャーテンプレート

弊社より発注させていただく際、ページャー部分のマークアップは以下に準拠してコーディングをお願いいたします。

  • < 前へ
  • 1
  • 2
  • 3
  • 4
  • …
  • 7
  • 次へ >

HTMLは下記のマークアップでお願いします。

<div class="pageList">
	<ul>
		<li><a href="#" class="prev">< 前へ</a></li>
		<li><a href="#">1</a></li>
		<li><span class="current">2</span></li>
		<li><a href="#">3</a></li>
		<li><a href="#">4</a></li>
		<li><span class="dots">…</span></li>
		<li><a href="#">7</a></li>
		<li><a href="#" class="next">次へ ></a></li>
	</ul>
</div>

CSSについては、以下をベースに変更をお願いします。

.pageList {
	text-align: center;
}

.pageList li {
	display: inline;
}

.pageList a,
.pageList span {
	text-decoration: none;
	border: 1px solid #95847B;
	padding: 5px 8px;
	margin: 0 5px;

	color: #95847B;
}

.pageList a:hover {
	background: #95847B;
	color: #FFF;
}

.pageList span.current {
	background: #DFDAD7;
	color: #663300;
	font-weight: bold;
}

.pageListi span.current {
	font-weight: bold;
}

.pageList span.dots {
	border: none;
}

また、各項目がパイプで区切られているデザインの場合、CSSはこちらを参照ください。
HTMLのマークアップは上と同じです。

  • < 前へ
  • 1
  • 2
  • 3
  • 4
  • …
  • 7
  • 次へ >
.pageList {
	clear: both;
}

.pageList a,
.pageList span {
	text-decoration: none;
	border-right: 1px solid #BFBFBF;
	padding: 3px 11px 3px 8px;
	margin: 0 2px;
}

.pageList a:hover {
	text-decoration: underline;
}

.pageList a.next {
	border: none;
}

.pageList span.current {
	font-weight: bold;
}
